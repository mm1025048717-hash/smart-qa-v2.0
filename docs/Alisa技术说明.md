# Alisa 核心技术说明

## 1. 技术架构

### 1.1 核心原理
ChatBI 基于**知识图谱**和**深度学习**技术，通过问答方式生成 SQL 查询。核心优势：

- **SemanticDB（语义数据库）**：100% 自研，为企业建立业务知识图谱
- **NL2Logicform（自然语言理解模块）**：100% 自研，将自然语言转化为 Logicform 中间表达

### 1.2 工作流程

```
用户提问 → NL2Logicform 理解意图 → 生成 Logicform → SemanticDB 转化为 SQL → 返回结果
```

**核心优势：**
- ⚡ **查询速度**：比其他 AI 快 3-5 倍
- 🎯 **准确率**：高达 99.8%
- 🔒 **数据安全**：支持纯内网部署，无需外网大模型
- 🚀 **零训练**：预训练完成，客户无需再次训练

## 2. SemanticDB 语义数据库

### 2.1 核心概念
- **实体（Entity）**：代表具体存在的物件和概念（如：门店、产品、人员）
- **事件（Event）**：围绕实体发生的具体事件，必须有时间戳（如：销售事件、入库事件）

### 2.2 知识图谱
通过业务建模，将数据模型转化为业务知识图谱，使系统理解企业业务。

### 2.3 Logicform
JSON 格式的数据库查询中间表达，架起自然语言和数据库查询语言的桥梁。

**示例：**
```json
{
  "schema": "sales",
  "query": { "时间": { "year": 2022 } },
  "preds": [{ "operator": "$sum", "pred": "销售额" }]
}
```

## 3. 数据安全

### 3.1 对外数据泄露
- ✅ 支持私有化部署
- ✅ 支持纯内网运行
- ✅ 无需将数据发送至外网

### 3.2 对内数据权限
- **表级别**：控制 Schema 访问权限
- **行级别**：通过筛选条件控制数据行
- **列级别**：控制指标访问权限
